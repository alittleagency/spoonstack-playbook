THE VISION
vision statement
the concept
Like a Lisa Frank planner met a Muji notebook and had a very organized baby.
Bold enough to be joyful. Structured enough to be useful. Warm enough to be safe.
what this means in practice
phase 1 was
a whisper — warm, restrained, almost invisible design. The interface steps back and the content breathes.
phase 2 is
a speaking voice — the same warmth, but bolder, more graphic, more confident. Color-blocked. Tactile. Physical.
the through-line
Same warm cream base. Same gold as primary accent. Same DM Sans + IBM Plex Mono. Same copy rules. Same voice. Different texture.
the inspiration
90s graphic design energy — Clueless color palettes, Lisa Frank boldness, Muji organization discipline. Physical, tactile, joyful.

COLOR SYSTEM
color blocking examples

Phase 2 uses colors in graphic blocks — two colors side by side as flat fills, not subtle gradients. The borders and offset shadows make each block feel physical, like sticker sheets.

dusty rose + sage green — joyful, balanced
butter yellow + cream — warm accent on neutral
periwinkle + ink — cool widget card on dark base
gold + ink — same as phase 1, bolder expression
how to use color blocks
section headers
full-width color fill behind a section header — one of the phase 2 palette colors
widget cards
accent bar becomes 4–5px and uses the more saturated phase 2 colors
callout blocks
full background fill in a phase 2 tint — not just a border, the whole area fills
never gradient
no gradients in phase 2 — always flat fills. the 90s were flat.
FONTS
typography evolution
same fonts, more expressive
DM Sans weights
Phase 2 uses 700–800 for display text. Tighter tracking (-0.04em to -0.06em). Bigger, more confident type scale.
IBM Plex Mono
Bold weight (700) for tags and chips in phase 2. More graphic, less metadata-like.
type scale
Phase 2 display: 42–48px at weight 800. Section titles: 26px weight 700. More expressive range than phase 1.
what stays
DM Sans for body/headings. IBM Plex Mono for meta/tags. 14–15px body size. 1.65 line height for body.
your brain.
built for you.
Phase 2 display type — DM Sans 800, tight tracking, two-color split.

MOTION
animation
phase 2 animation spec
default transition
120ms (vs. 200ms in phase 1) — snappier, more physical
button press
transform: translate(3px,3px); box-shadow: none; — 80ms duration. presses in, then springs back.
spring curve
Bouncy spring on press: scale 0.92 → 1.04 → 1.0. Things feel physical, like they have weight.
sheet entrance
Bottom sheet slides in with slight overshoot — 8px past target, then snaps to position. 160ms total.
card tap
Brief scale 0.97 on tap, spring back. Under 100ms. Immediate tactile response.
no fades
Prefer translate/scale transitions over opacity fades. Phase 2 things move — they don't fade.
spring animation spec
/* button press spring */
@keyframes btnPress {
  0%   { transform: translate(0,0) scale(1); }
  40%  { transform: translate(3px,3px) scale(0.97); box-shadow: none; }
  70%  { transform: translate(-1px,-1px) scale(1.02); }
  100% { transform: translate(0,0) scale(1); box-shadow: 3px 3px 0 var(--ink); }
}

/* snappy default */
transition: all 0.12s ease-out;

/* overshoot entrance (bottom sheet) */
animation: slideUp 0.16s cubic-bezier(0.34, 1.56, 0.64, 1);
SUMMARY
what stays · what changes
stays the same
  • spoonstack always lowercase
  • DM Sans + IBM Plex Mono
  • 4-tab nav structure
  • cream/warm base (#FAFAF7)
  • gold as the primary accent
  • voice and copy rules
  • no em dashes, no exclamation marks
  • no clinical language
  • ND design principles
  • 4 home widget cards (same ones)
  • capture bar in dark (#1A1714)
what changes
  • buttons get 1.5px borders + hard offset shadows
  • colors get more saturated + color-blocked
  • typography gets bigger and bolder (800 weight)
  • animation gets bouncier (spring physics)
  • cards get thicker borders + some offset shadows
  • tags get borders + offset shadows too
  • icons become slightly hand-drawn / organic
  • 5 new colors in the palette
  • transitions faster (120ms vs 200ms)
  • section headers can use color-block fills
the guiding principle for phase 2

Phase 2 is still spoonstack — it just speaks up. The warmth, the restraint, the care for ND brains — all of that stays. What changes is the graphic confidence. Phase 1 was designed to disappear. Phase 2 is designed to delight. But never at the expense of safety, clarity, or the people it's built for.
